# Object Store  

## Objective  
To create an object storage service where user can upload files via an API/UI  

## Future Features  
- Distributed system with backup and load balancing
- Object Router

## Tech Stack:  
- Backend: Python, Flask, SQLite  
- Frontend: Javascript  
- Infrastructure: Docker
